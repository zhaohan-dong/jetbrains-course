You need to create a class of Users for your project. Here are some attributes that you'll need:

active — a boolean attribute that indicates whether a user is active at the moment,

n_active — the number of active users,

user_name — string with the username,

users — a list of all users.

Think about which attributes should be class attributes and which should be instance ones. Then create that class below.

If there are 2 or more instance attributes, in the __init__() method, list them in alphabetical order!

NOTE: in this task, you just need to define the class. You do NOT need to create an instance or print anything. You are also NOT required to write any additional code (for example, for modifying class attributes).
