Bittersweet occasion Open on JetBrains Academy

Mysterious things have happened on the Hogwarts Express! Someone enchanted a food trolley and all the goodies got mixed up on the shelves. While the trolley witch is looking for the culprit, you will place Drinks, Pastry and Sweets on three respective shelves.

These base classes have various children, for example, PumpkinJuice inherits from Drinks. Find a parent class for an unknown object stored in the variable child and print out the parent's name.


Sample Input:
PumpkinJuice
Sample Output:
Drinks

Sample Input:
CauldronCake
Sample Output:
Pastry

Sample Input:
ChocolateFrog
Sample Output:
Sweets
